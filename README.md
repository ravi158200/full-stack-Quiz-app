MERN Quiz App Description

The MERN Quiz App is a full-stack web application built using the MERN stack (MongoDB, Express.js, React.js, and Node.js). This application allows users to take quizzes on different topics and test their knowledge through multiple-choice questions.

The frontend is developed using React.js, providing a responsive and interactive user interface. The backend is built with Node.js and Express.js, which handles API requests, quiz logic, and user authentication. MongoDB is used as the database to store quiz questions, user data, and results.

Users can select a quiz, answer multiple-choice questions, and submit their responses to view their final score. The system calculates results automatically and displays performance instantly.

Key Features

User-friendly quiz interface

Multiple-choice questions

Real-time score calculation

REST API integration

MongoDB database for storing quiz data

Responsive design

Technologies Used

MongoDB – Database

Express.js – Backend framework

React.js – Frontend UI

Node.js – Server environment
